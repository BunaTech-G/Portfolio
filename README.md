# Portfolio (HTML/CSS)

Portfolio one-page (Accueil / À propos / Projets / Contact) avec une ambiance dark et un accent orange.

## Fichiers

- `index.html` : structure et contenu
- `style.css` : styles (thème dark + orange, responsive)
- `assets/bg.svg` : fond immersif (local)

## À personnaliser (rapide)

Dans `index.html` :

- Nom / branding : section header (marque "Buna")
- Texte d’intro : section `#accueil`
- Email : remplace `buna.dev@email.com`
- GitHub : remplace `https://github.com/TON-USERNAME`
- Projets : remplace les cartes dans la section `#projets` (liens Démo/Code)

## Idée d’évolution

- Ajouter un formulaire de contact (JavaScript)
- Ajouter des animations légères (en respectant `prefers-reduced-motion`)
- Ajouter une page projet détaillée plus tard si besoin
